# Java Development Training Course

Dieses Repository enthält meinen vollständigen Lernfortschritt aus dem Java-Modul meiner Umschulung zur Fachinformatikerin (WBS Training).

### Inhalte & Technologien:
* **Grundlagen:** Datentypen, Schleifen, Bedingungen.
* **Objektorientierte Programmierung (OOP):** Klassen, Vererbung, Polymorphismus.
* **Datenstrukturen:** Collections, Listen, Arrays.
* **Algorithmen:** Übungsaufgaben (z.B. Tennis Game, Logik-Aufgaben).

Der Code ist chronologisch nach Kurstagen ("Tag") organisiert und zeigt meine Entwicklung von den Grundlagen bis zu komplexeren Strukturen.
